This Site Is Available Under the V4V Model

The term “Value for Value” originated on the No Agenda show. The idea behind the Value for Value model is simple: I put work into creating this site. Have you received any value from it? If you have, then please contribute that value back to the site.

There’s many different ways to contribute back to Value for Value projects. For this one in particular, here’s what you can do to contribute value:

  • Link to this site
  • Send feedback (bug reports, suggestions, etc.). If that link doesn’t work, send it to “Jason Yundt <jason@jasonyundt.email>”.
  • Submit patches
